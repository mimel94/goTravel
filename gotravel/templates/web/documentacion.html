Guia de referencia tecnica



Como decargar y configurar el proyecto

  • 1) Descargamos el proyecto en github en el siguiente enlace goTravel project
  • 2) Una vez descargado debemos montar un ambiente de desarrollo bajo python version 3.8
  • 3) una vez en el ambiente debemos instalar los requerimientos del sistema con el siguiente comando

    • pip install -r requirements.txt

  • 4) una vez instalado todos los requerimientos, nos ubicamos en la raiz del proyecto y levantamos el servidor de Django

    • python manage.py runserver
